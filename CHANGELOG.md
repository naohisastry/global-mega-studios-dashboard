# Changelog

## [1.0.0] - 2026-07-06

### Added

- Initial public version of the Global Mega-Studios / Super-Indies Financial Dashboard.
- Added interactive dashboard covering major global and European production studio groups from 2015 to 2025.
- Added financial comparison of Banijay, Fremantle, ITV Studios, BBC Studios, Mediawan, and All3Media.
- Added revenue trend visualization.
- Added EBITA / EBITDA margin comparison.
- Added business model mix analysis.
- Added M&A timeline and ownership change visualization.
- Added company selector and currency mode switching.
- Added Chart.js-based interactive visualizations.
- Added structured financial and company data in `data.js`.
- Added dashboard logic and visualization control in `app.js`.

### Documentation

- Added bilingual README summary in English and Japanese.
- Added `DATA_SOURCES.md` to summarize source categories, financial disclosures, M&A references, and evidence structure.
- Added `METHODOLOGY.md` to explain data definitions, assumptions, limitations, currency treatment, and analytical scope.

### Notes

- Some figures are based on official company disclosures.
- Some figures are based on parent-company segment disclosures.
- Some figures are based on public reports, market research estimates, and analyst-derived assumptions.
- Profitability metrics may differ across companies, including EBITA, EBITDA, adjusted EBITDA, operating profit, and segment profit.
- Currency conversion may affect cross-company comparison.
- M&A activity may create discontinuities in time-series data.
